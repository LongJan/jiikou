## jiikou

jiikou是`実行(じっこう)`的罗马拼音，意思是执行。故名思义，这个kotlin库有两个功能
- 一个让你的jar包变成可执行文件的命令行工具，比如`happy.jar`可以直接`sh happy.jar`或者`./happy.jar`
- 提供api可以让你在你的程序中简单地把jar包变为可执行的文件
